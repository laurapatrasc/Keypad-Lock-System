# Keypad-Lock-System
Proiectul constă în crearea unui sistem de securitate care permite utilizatorului să deblocheze sau să blocheze un dispozitiv prin intermediul unei tastaturi matriciale 4x4. La inițializare, sistemul afișează „Sistem Inactiv” pe un LCD 1602. Utilizatorul introduce un cod prestabilit, iar dacă este corect, statusul devine „Deblocat”. Apoi LCD-ul și aplicația mobilă, conectată prin Bluetooth se actualizează. În caz de cod greșit, apare mesajul „Cod Incorect”. Sistemul poate fi, de asemenea, blocat printr-un buton de pe keypad, actualizând din nou statusul pe LCD și aplicație. Comunicarea dintre STM32, LCD și aplicația mobilă asigură sincronizarea stării sistemului în timp real.
